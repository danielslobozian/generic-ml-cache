> [!TIP]
> This page explains when the pattern is useful and how it relates to gmlcache’s exact replay model.

A caller may run similar detached requests across different adapters, models, or
effort levels.

gmlcache treats those as different execution requests. It can record each result,
usage envelope, generated files, and replay behavior.

The documentation should not claim superiority over other tools or providers.
This use case simply explains that gmlcache can help callers inspect their own
experiments.
