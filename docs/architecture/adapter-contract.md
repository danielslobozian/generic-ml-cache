---

An adapter translates a gmlcache execution request into one backend call and then
translates the backend result into the common response model.

An adapter should provide:

- launch construction,
- prompt/context delivery,
- grant mapping,
- structured output parsing where available,
- clean answer extraction,
- normalized usage extraction,
- raw usage preservation,
- clear unsupported-feature behavior.
